### 病害标注样式（2026-03-21）
- 第一段：从病害区域引出，45度角，长度8
- 第二段：水平线，长度根据文字长度决定

**文字位置**：
- 病害名（如"剥落"）：在第二段水平线上方，紧贴线条
- 数值（如"S=0.02m²"、"L=4.00m"）：在第二段水平线下方
- 字体大小：2.5
- 颜色：7

**示例**（左翼缘板剥落 x=0.4-0.6m, y=0.5-0.6m）：
- 起点：(89.8, 267.8) - 区域右上角
- 折点：(95.5, 273.5) - 45度角，长度8
- 终点：(107.5, 273.5) - 水平线，长度约12
- "剥落"位置：(97.5, 274.8) - 线上方
- "S=0.02m²"位置：(97.5, 269.8) - 线下方

**裂缝标注**：
- 折线引线样式同剥落
- 第二段线上方：病害名 + 长度，如"纵向裂缝 L=4.00m"
- 第二段线下方：宽度，如"W=0.10mm"

**网状裂缝标注**：
- 病害区域用波浪线填充（波长4，波幅0.3，线间距1）
- 折线引线样式同剥落
- 第二段线上方：病害名，如"网状裂缝"
- 第二段线下方：面积，如"S=9.00m²"

**裂缝群绘制（N=数字多条裂缝）**：✅
- 所有含 N=数字 的裂缝病害，都使用手绘风格绘制多条裂缝线
- 裂缝间距：线间距 1 CAD单位，线条位于区域中线（+0.5偏移）
- 手绘风格：
  - 使用正弦波模拟自然曲线：`math.sin((x - x_start) / 3) * 0.15`
  - 添加微小随机偏移：`random.uniform(-0.1, 0.1)`
  - 步进0.8生成点，最后补终点
- 标注样式：折线引线 + 第二段线上方`病害名 L总=xxxm N=数字条`，下方`间距xxxm`
  - L总从Excel读取 L总=xxxm 字段
  - 间距从Excel读取 间距xxxm 字段

**剥落露筋标注**：
- 病害区域先画白色剥落图（来自"剥落、掉角.dxf"）
- 再叠加红色虚线格栅（3条横线 + 3条竖线）
- 折线引线样式同剥落
- 第二段线上方：病害名，如"剥落露筋"
- 第二段线下方：面积，如"S=0.02m²"

### 模板文字替换方法
**重要**：模板中的占位符（如 LLL、QQQ、KKK、HHH、a-a、b-b）可能是 **TEXT** 或 **MTEXT** 类型：
- **TEXT** 类型：使用 `entity.dxf.text` 属性
- **MTEXT** 类型：使用 `entity.text` 属性（无 .dxf）
- MTEXT 可能带格式代码，如 `{\C0;LLL}`

**替换函数**：
```python
def update_text_in_msp(msp, old_text: str, new_text: str):
    for entity in msp:
        if entity.dxftype() == 'TEXT':
            if old_text in entity.dxf.text:
                entity.dxf.text = entity.dxf.text.replace(old_text, new_text)
        elif entity.dxftype() == 'MTEXT':
            content = entity.text
            if old_text in content:
                entity.text = content.replace(old_text, new_text)
```

### 剥落露筋绘制（2026-03-22 最终确定，用户确认）✅
- = **剥落不规则图形（来自 `剥落、掉角.dxf` 的 `adfsd` 块）** + **叠加红色虚线格栅（3条横线+3条竖线）**

- `draw_peel_off()` 增加 `return_bbox` 参数，返回 (ax1, ay1, ax2, ay2)
- 注意：`load_rebar_geometry()` 函数保留但不再使用
