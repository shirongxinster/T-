## 单柱墩平铺图例（蜂窝/麻面/剥落）加入1度倾斜偏移（最终正确版本）

**核心原则：**
- 坐标转换不加倾斜，保持原样：`cad_y = origin_y - y * scale_y`
- `cad_y_max`（上边界）= `origin_y - y_min * scale_y`，作为 `start_y`
- 图例从顶部向下画：变换公式用 `dy = tile_y - src_max_y * scale`（用src_max_y，不是src_min_y）
- tile_y 代表图例**顶部**目标Y，向下平铺：`tile_y = start_y - row * scaled_h - col_y_offset`
- 倾斜偏移：`col_y_offset = col * scaled_w * tan(rotation_deg)`，右侧列Y更小

**验证：** 7号盖梁大桩号面麻面(x=6～10, y=0～0.1)：
- 目标区域 Y=262.7～264.0，上边界=264（桥面），向下画 ✓
- 不再飘到桥面以上

**错误教训：**
- 错误1：倾斜偏移加到坐标转换里 → 图例飘到桥面以上
- 错误2：用`src_min_y`做变换(图例底部对齐) → 图例往上画超出边界
- 正确：用`src_max_y`做变换(图例顶部对齐) + 向下铺

**实现：**
- 文件：bridge_disease_main_lower.py
- 新增函数：draw_pier_mesh_crack (约第2810行)
- 修改：process_single_pier_disease 的柱子处理分支 (约第2630行)

**验证：** 4-1号.dxf 文件包含 18 条 LWPOLYLINE（波浪线），网状裂缝正确绘制。

## 单柱墩盖梁和墩柱配对逻辑

**问题：** 4-1号墩柱被当作独立构件处理，没有和4号盖梁画到一张图里。

**原因：** 单柱墩处理逻辑直接遍历 `comp_id`，没有区分盖梁和墩柱，也没有配对。

**解决方案：** 添加类似双柱墩的配对逻辑：
1. 分离盖梁病害和墩柱病害（按 `缺损位置` 判断）
2. 使用 `get_cap_base()` 提取主编号（如 "4-1号" -> "4"）
3. 按主编号配对：盖梁 "4号" + 墩柱 "4-1号" 画到同一张图
4. 文件名使用盖梁号（如 `单柱墩_4号.dxf`）

**实现：** 修改单柱墩处理部分（约第3198行）
