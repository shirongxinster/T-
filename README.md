# 桥梁病害CAD标注系统

根据桥梁病害Excel记录自动生成CAD病害展示图的工具。

## 功能概述

- 读取Excel中的桥梁病害记录
- 根据构件类型匹配对应的CAD模板（T梁、墩柱、桥台）
- 自动将病害标注到对应的CAD图纸上
- 支持多种病害类型（裂缝、剥落、网状裂缝、锈胀露筋等）

## 文件结构

```
├── bridge_disease_main.py      # 主程序
├── templates/                  # 模板目录
│   ├── 构件/                   # 构件模板
│   │   ├── 40mT梁.dwg

│   │   ├── 单柱墩.dwg
│   │   ├── 不带台身桥台.dwg
│   │   └── 带台身桥台.dwg
│   └── 病害图例/               # 病害图例
│       ├── 裂缝及其长宽.dwg
│       ├── 网状裂缝.dwg
│       ├── 剥落、掉角.dwg

```

## 使用方法

1. 确保已安装浩辰CAD
2. 准备病害Excel文件，格式如下：
   - 第1行：路线名称（如：路线名称: G11鹤大高速）
   - 第2行：桥梁名称（如：桥梁名称：K572+774红石牡丹江大桥（右幅））
   - 第4行：构件类型（如：上部（40mT梁））
   - 后面是病害数据行：序号、构件编号、构件类型、病害描述

3. 运行程序：
   ```bash
   python bridge_disease_main.py [excel文件路径]
   ```

## 病害Excel格式示例

| 序号 | 构件编号 | 构件类型 | 病害及程度 |
|------|----------|----------|------------|
| 1 | 1-1号 | T梁 | 梁底，x=10～14m，y=0.5m，纵向裂缝 L=4.00m，W=0.10mm |
| 2 | 1-2号 | T梁 | 左翼缘板，x=0.4～0.6m，y=0.5～0.6m，剥落 S=0.02m2 |

## 坐标系统

根据模板中的标尺体系进行坐标转换：
- 标尺刻度1m = 10个CAD单位
- 不同构件类型有不同的坐标系

## 依赖

- Python 3.x
- pywin32 (用于COM接口)
- pandas (用于读取Excel)
- 浩辰CAD（需要安装并运行）

## 注意事项

1. 泛白病害和马蹄部位的病害不处理
2. T梁每张图放2个构件
3. 墩柱和桥台每张图放1个构件
4. 所有图纸纵向排列，不会重叠
